# Installation

## Basic install

=== "pip"

    ```bash
    pip install langchain-adk
    ```

=== "uv"

    ```bash
    uv add langchain-adk
    ```

## With an LLM provider

=== "OpenAI"

    ```bash
    pip install langchain-adk[openai]
    ```

=== "Anthropic"

    ```bash
    pip install langchain-adk[anthropic]
    ```

=== "Google Gemini"

    ```bash
    pip install langchain-adk[google]
    ```

=== "MCP"

    ```bash
    pip install langchain-adk[mcp]
    ```

!!! info "Requirements"
    **Python >= 3.10** is required.
